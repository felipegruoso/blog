## Blog ##

This is a web-application that simulates a blog.

This project was created to be used as demonstration on a Ruby on Rails minicourse in Garça-SP (FATEC).

### Running the application

To run this application is necessary to follow these instructions:

1. Clone the repository.
2. Run `bundle install`
3. Run `bundle exec rake db:create`
4. Run `bundle exec rake db:migrate`
5. Run `rails s`
